# README

Jonathan Goldsmith, the XX guy, needs to update the style of his 'i Beer' blog. His current blog has no styling whatsoever. 

Style the given HTML document to match the 'blog' mockup using the following:

- Positioning
- Z-index
- Flexbox
- Adjacent values
